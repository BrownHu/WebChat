# Go实现的聊天室


### 安装使用

```
    git clone https://github.com/BrownHu/WebChat 
    
    export  GOPATH=/WebCat
    
    dep ensure
    
    bee run 
    
    
```

### 开始使用

```apple js
    访问localhost:3000 
```
